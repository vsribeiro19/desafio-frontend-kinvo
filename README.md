## Instalação

Clone o repositório
``` bash
git clone https://github.com/vsribeiro19/desafio-frontend-kinvo.git
```
rode o seguinte comando para instalar a biblioteca de gráficos (charts.js)
```bash
npm install
```
por fim, execute o arquivo **index.html**
